Troubleshooting Guide

            

                
If all tests fail with 500 errors:

                
    
                    
  • Check Laravel application is running properly
    • 
                    
  • Verify database connection in aprati_backend/.env
    • 
                    
  • Check Laravel logs at aprati_backend/storage/logs/laravel.log
    • 
                    
  • Run: php artisan config:clear && php artisan cache:clear
    • 
                


                
If tests fail with 404 errors:

                
    
                    
  • Check .htaccess file in root directory has correct rewrite rules
    • 
                    
  • Verify routes exist in aprati_backend/routes/api.php
    • 
                    
  • Make sure mod_rewrite is enabled on server
    • 
                


                
If some tests pass but others fail:

                
    
                    
  • Specific API endpoints may have errors in their controllers
    • 
                    
  • Check individual controller files in aprati_backend/app/Http/Controllers/Api/
    • 
                    
  • Database tables for failing endpoints may be missing data
    •
