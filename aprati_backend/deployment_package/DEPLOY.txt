COMMON ISSUES AND FIXES:
================================================

Issue: Still getting 404 errors after running script
Fix: Check Apache/Nginx configuration
  - Document root MUST point to: /path/to/aprati_backend/public
  - NOT to: /path/to/aprati_backend

Issue: "Internal Server Error" instead of 404
Fix: Check file permissions
  chmod -R 755 storage bootstrap/cache
  chmod -R 644 .env

Issue: Routes work locally but not on server
Fix: Enable mod_rewrite (Apache)
  sudo a2enmod rewrite
  sudo service apache2 restart

Issue: .htaccess not working
Fix: Enable AllowOverride in Apache config
  In /etc/apache2/sites-available/your-site.conf:
  <Directory /path/to/aprati_backend/public>
      AllowOverride All
  </Directory>
